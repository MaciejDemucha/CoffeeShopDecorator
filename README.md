# CoffeeShopDecorator

Implementacja wzorca projektowego Decorator na przykładzie systemu obsługującego kawiarnię oferującą zakup kawy lub herbaty z różnymi dodatkami. Napoje i dodatki z menu reprezentowane są jako klasy:
- Coffee
- Tea
- Cream
- Caramel
- Chocolate
- Honey
